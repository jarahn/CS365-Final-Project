Welcome to our CS365 guide

CS365 - Client Side Web Development - is a class offered to Northern Michigan University students and required for the mobile application and web development major.
*Prerequisites: CS201 or both CS101 and CS122
Course Description: This course teaches how to make fully interactive websites and web applications, targeted to multiple devices. Topics include HTML5, CSS3, JavaScript, jQuery and Node js.

This guide will serve as an overview to CS365 as well as a reference to some of the technologies learned in class.

What Is Client Side Web Development?

In simple terms, it's the programming of everything a user sees on a website. The user facing system or "front end" of a website is referred to in the industry as a client side application. A client side application is the program that runs on a users machine, in a web browser.
The two main functions of a client side program are to:
1) Provide a styled interface between the user and the server. Usually with heavy emphasis on the style.
2) Interact with local storage (cookies, etc.)

Client side applications are generally made up of three main tools
-HTML(5): A markup language used to mark and describe content to achieve the proper display in web browsers.
-CSS(3): A style sheet language used to describe the desired presentation of information written in a markup language (like HTML).
-JavaScript: A high level, interpreted, scripting language used to control the behavior of elements.
In Short: HTML provides the content for the webpage, CSS provides the styling, and JavaScript provides the behavior of elements.

Back End - Servers/Server Side

Although it is not a main focus of this course, server technologies such as node.js, socket.io, and AJAX are covered. Servers are used to store information which can be communicated with a user rather than given to them directly in the client side application. They also allow for multiple client connections and facilitate keeping data consistent and correct across clients. This site uses a server to process user requests and interact with a databse of ebooks available through NMU.
