MongoDB

An open source database solution

MongoDB is lightweight easy to implement database. It is an open source software that offers commercial support. Generally used for document storage.

setup

Mongo db has multiple version, the two most prominent being the community version and enterprise version the community version is the free version that we want. To download the community version click here. In order to get mongo running prorperly we have to add the mongo directory to the windows filepath so keep track of whree you saved mongo to. This allows for us to run mongo commands from the cmd line.

Adding to filepath(on windows)

1. Open your file explorer and find the entry "This PC or My Computer", right click it and open the properties menu 2. Once in the properties menu, open the "advanced system settings" menu. You should see a button named "environment variables" click on it to open the menu. 3. You should see a display box with file paths like so click on the entry titled "path" and hit the edit button. Once the edit environment variables window is open click underneath the very last entry on the list, then select new from the right hand side of the screen. add the path to your newly downloaded mongoDB directory here.
