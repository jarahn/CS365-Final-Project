MongoDB


An open source database solution

MongoDB is an open source database software produced by MongoDB inc (formerly known as 10gen). While technically being open source, the company still offers an enterprise package. It is a lightweight "NoSQL" document database that uses "JSON-like" entries to store data. This json like format allows for database queries to be simpler than other formats.

Setup

Mongo db has multiple version, the two most prominent being the community version and enterprise version the community version is the free version that we want. To download the community version click here. In order to get mongo running prorperly we have to add the mongo directory to the windows filepath so keep track of whree you saved mongo to. This allows for us to run mongo commands from the cmd line.

Adding to Filepath(on windows)

1. Open your file explorer and find the entry "This PC or My Computer", right click it and open the properties menu

2. Once in the properties menu, open the "advanced system settings" menu. You should see a button named "environment variables" click on it to open the menu.

3. You should see a display box with file paths like so click on the entry titled "path" and hit the edit button. Once the "edit environment variables" window is open, click underneath the very last entry on the list, then select new from the right hand side of the screen. add the path to your newly downloaded mongoDB directory here. Once you've added the path hit okay. You should now be able to run the mongo server!

Database Structure

Mongo db entries are modeled like so: A database has collections, a collection contains either one or more documents, these documents will contain fields. This modeling allows for accessing data similar to accessing an objects property in some programming lagnuages.

Creating and running a database

In order to run the mongo shell, type "mongod" in the command prompt(this will only work after the steps above). Once in the mongo shell type "use databaseName" to either create a database or to use an existing database.
