Vue

Vue(pronounced "view") is a progressive javascript framework for building user interfaces on single page applications. The core library is focused on the view layer, with a model-view viewmodel. It is often revered as easy to pick up and integrate with other libraries or existing projects. Companies such as Adobe, Alibaba, and Grammarly have all used Vue in some form for their web applications.

Set Up Instructions

1. Click the Vue logo above for redirection to the vue.org webpage.

2. Click the green installation button on the homepage.

3. Scroll down and choose the Vue package that suits your needs.

-There are two versions available for download, production and development. They provide the same functionality, but the development version contains error and warning messages for common pitfalls made by programmers. Naturally, the development version can take up uneeded space in a functioning production environment. The production version, while beneficial, is unnecessary for CS365.

4. After installing, move a copy of 'vue.js' into the public folder of your project directory.

Important Notes

* You must include the vue.js script in your html file to use the vue framework.

* Vue is not reactive to individual array elements. - one workaround is to use the splice method whenever you update an array.

* You must put "this." before the name of a data element when acessing it from either the methods or computed sections. (See example below)

* The element in the html file you are connecting to is placed at the top following "el:"

How It's Made:

To reactively change the "src" of an image element with Vue, all you need is "v-bind:src=". Replacing the normal image source with a data element (string) found in your Vue object.
