Vue

Vue(pronounced "view") is a progressive javascript framework for building user interfaces on single page applications. The core library is focused on the view layer, with a model-view viewmodel. It is often revered as easy to pick up and integrate with other libraries or existing projects. Companies such as Adobe, Alibaba, and Grammarly have all used Vue in some form for their web applications.

Important Notes

* You must include the vue.js script in your html file to use the vue framework.

* Vue is not reactive to individual array elements. - one workaround is to use the splice method whenever you update an array.

* You must put "this." before the name of a data element when acessing it from either the methods or computed sections. (See example below)

* The element in the html file you are connecting to is placed at the top following "el:"
