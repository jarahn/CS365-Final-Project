Welcome to our CS365 guide

CS365 - Client Side Web Development - is a class offered to Northern Michigan University students and required for the mobile application and web development major.

This guide will serve as an overview to CS365 as well as a reference to the technologies learned in class.

What is client side web development?

A client side application is the program that runs on the client machine (browser) and deals with the user interface/display and any other processing that can happen on a client machine.
The main functions of a client side program are:
1) Interact with temporary storage
2) Make interactive web pages
3) Interact with local storage
4) Sending request for data to server
5) Send request to server
6) work as an interface between server and user

Servers

Servers are used to "store" information. They allow for multiple client connections and facilitate keeping data consistent and correct across clients.
